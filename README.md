# Agent SOP

A library of standard operating procedures and templates for Claude Code agent sessions.

## What this is

Claude Code agents work best when they have consistent structure across sessions and projects. This library defines a standard file set, additive-only update rules, session checklists, and templates — so every agent starts every session with full context.

## Structure

- `docs/sop/` — Core SOP documents
- `docs/templates/` — Ready-to-use templates for each standard file
- `docs/examples/` — Walkthrough guides for new and existing projects
- `docs/build-plans/` — Phase-level build tracking

## Status
